## 🌐 Mis principios de desarrollo

| 💡 Principio | Descripción |
|---|---|
| **Clean Code** | Código legible, mantenible y sin malas sorpresas |
| **REST First** | APIs bien diseñadas que cualquiera pueda consumir |
| **Containerize Everything** | Docker para ambientes reproducibles |
| **Version Control** | Historial de Git que cuenta la historia del proyecto |
| **Test Early** | Pruebas desde el inicio, no al final |
